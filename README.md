### 表
* **counters**
  * 自增计数器
* **users**
  * 用户
* **sessions**
  * sessions，关联登录用户
* **questions** 
  * 问题，关联题主
* **answers** 
  * 答案，关联答主和归属问题
* **qfollow**
  * **users** 和 **questions** 的映射
* **ufllow**
  * **users** 和 **users** 的映射
